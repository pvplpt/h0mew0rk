Добавил классы и проверку диагоналей.

Для сброки и запуска проекта, можно выполнить в папке TicTacToe:
```bash
mkdir build  
cd build  
cmake ..  
cmake --build .  
./src/tictactoe  
```
> **Замечание:** В теминале ОС Windows переключение на кодировку utf-8 можно выполнить командой: CHCP 65001
